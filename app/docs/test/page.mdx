## Introduction

ScoutMe allows you to **track**, **analyze**, and **compare** football players easily.  
It’s designed for scouts, clubs, and agents who need fast access to data and performance metrics.

You can use ScoutMe to:
- Store player profiles
- Record match stats
- Filter by position, club, or potential rating

[Learn more about features →](#advanced-settings)

## Installation

To install the app locally, run:

```bash
npm install scoutme
npm run dev
```
