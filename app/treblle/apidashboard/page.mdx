# API Dashboard

The API Dashboard in Treblle gives you a 360-degree, bird's-eye view of all your APIs in one place, helping you track, manage, and improve your API estate with clarity and speed. 

It goes a level deeper than the Home page, providing detailed insights so everyone, from someone like a CIO or a CTO to a developer or even a non-technical person, can clearly understand the API landscape.

</InfoCard>

### Individual API Blocks

Each API in your workspace is displayed as a **dedicated, clear block** showing essential details at a glance, making it easy to scan and prioritize across all your APIs.

### Heartbeat with Good (Green) and Bad (Red) Requests

Each API block includes a **compact heartbeat graph** showing request trends with color-coded indicators:

- **Green** indicating successful requests
- **Red** indicating errors

<TipCard>
The heartbeat visualization makes it easy to spot spikes, drops, and anomalies in API performance at a glance, helping you quickly identify issues before they escalate.
</TipCard>
